# Hyperparameter Tuning

This project covers Gaussian Processes and Bayesian Optimization for hyperparameter tuning.

## Tasks

| Task | File | Description |
| --- | --- | --- |
| 0. Initialize Gaussian Process | [0-gp.py](0-gp.py) | Create the class `GaussianProcess` that represents a noiseless 1D Gaussian process |
| 1. Gaussian Process Prediction | [1-gp.py](1-gp.py) | Update `GaussianProcess` with a `predict` method |
| 2. Update Gaussian Process | [2-gp.py](2-gp.py) | Update `GaussianProcess` with an `update` method |
| 3. Initialize Bayesian Optimization | [3-bayes_opt.py](3-bayes_opt.py) | Create the class `BayesianOptimization` |
| 4. Bayesian Optimization - Acquisition | [4-bayes_opt.py](4-bayes_opt.py) | Update `BayesianOptimization` with an `acquisition` method |
| 5. Bayesian Optimization | [5-bayes_opt.py](5-bayes_opt.py) | Update `BayesianOptimization` with an `optimize` method |
| 6. Bayesian Optimization with GPyOpt | [6-bayes_opt.py](6-bayes_opt.py) | Optimize a machine learning model using GPyOpt |

## Learning Objectives
* What is a Gaussian Process?
* What is a kernel function?
* What is Bayesian Optimization?
* What is an acquisition function?
* What is Expected Improvement?
* What is GPy?
* What is GPyOpt?
