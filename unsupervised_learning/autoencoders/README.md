# Autoencoders

This project explores different types of autoencoders using TensorFlow and Keras.

## Tasks

### 0. "Vanilla" Autoencoder
- [0-vanilla.py](0-vanilla.py): Implements a basic autoencoder with dense layers.

### 1. Sparse Autoencoder
- [1-sparse.py](1-sparse.py): Implements a sparse autoencoder using L1 regularization on the latent space.

### 2. Convolutional Autoencoder
- [2-convolutional.py](2-convolutional.py): Implements a convolutional autoencoder for image data.

### 3. Variational Autoencoder
- [3-variational.py](3-variational.py): Implements a variational autoencoder (VAE) that Learns a latent probability distribution.
