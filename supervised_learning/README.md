# Alu-Machine-Learning

This repository contains various machine learning and mathematics projects completed as part of the ALU curriculum.

## Projects

### 1. Math
*   **Calculus**: Implementation of basic calculus concepts including summation, derivatives, and integrals in Python.
*   **Linear Algebra**: Foundational linear algebra operations and matrix manipulations.

### 2. Supervised Learning
*   **Classification**: A comprehensive implementation of binary and multiclass classification algorithms from scratch using NumPy.
    *   Single Neuron (Logistic Regression)
    *   Neural Network with one hidden layer
    *   Deep Neural Network (Multi-layer) with support for Sigmoid, Tanh, and Softmax activations.
    *   Implementation includes forward propagation, cost calculation, gradient descent, and training loops with visualization support.

## Requirements
*   Python 3.x
*   NumPy
*   Matplotlib (for training cost visualization)
*   Pycodestyle (for linting)
